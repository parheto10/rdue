Traitement du fichier excel

1 - Convertir le fichier excel en CSV
2 - Sélectionner toutes les occurrances de ',' et les remplacer par '.' (Possibble de faire cela avec Excel avant la conversion en CSV)
3 - Sélectionner tous les espèces qui existent avant et après';' (Possibble de faire cela avec Excel avant la conversion en CSV en supprimant tous les espaces avant et après les valeurs des cellules)
4 - Accorder les noms des espèces avec les libellés de leurs équivalents de la base de données. C'est aussi l'ocassion d'enregistrer en base les espèces qui n'y sont pas
5 - Réduire le nom de certaines colonnes pour éviter les caractères accentués.
